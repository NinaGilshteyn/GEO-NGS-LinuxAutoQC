AutoDataProcessing

Bash scripts for reformatting GEO downloads into analysis-ready CSV files

A set of optimized bash scripts for reformatting raw scRNA-seq data downloaded from NCBI GEO. The pipeline takes the standard .mtx matrix and .tsv barcode/feature files and converts them into a single, analysis-ready .csv file — eliminating the manual formatting overhead before downstream ScanPy analysis.

Bash Scripts

What the scripts do

01_download.sh

Fetches raw scRNA-seq data from NCBI GEO using accession IDs, extracting the compressed archive.

02_parse_mtx.sh

Reads the sparse .mtx matrix file and maps row/column indices to barcodes and feature names from the .tsv files.

03_merge.sh

Joins the parsed matrix with barcode and gene feature tables into a single unified data frame.

04_to_csv.sh

Exports the merged data frame as a labeled .csv file, ready for import into ScanPy or any downstream tool.

About this project

Background & motivation

Context

Developed during post-graduate research at the Deeds Lab (UCLA). The goal was to eliminate the tedious manual step of reformatting GEO downloads before they can be loaded into ScanPy.

Tools & environment

Bash, Python 3, ScanPy, NCBI GEO. Handles the standard 10x Genomics sparse format: matrix.mtx, barcodes.tsv, and features.tsv.
